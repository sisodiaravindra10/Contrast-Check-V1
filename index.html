About this artifact

A one-page case for calibrated dark mode. The shell uses a warm slate palette, one of the six. The argument is the design.

Built as a single static HTML file. No build step, no tracking, no dependencies. Open it from a USB stick if you want.

Methodology

Contrast ratios computed from OKLCH coordinates via OKLab to linear sRGB to WCAG 2.1 relative luminance. APCA Lc per the Myndex reference algorithm.

AA: 4.5:1 body, 3:1 large. AAA: 7:1 body, 4.5:1 large. APCA Lc 75+ is body-readable.
